# Field Operations

Flutter mobile app for secure field-operations workflows, starting with the UI foundation and mock navigation.

## Current Scope

- Splash screen
- Secure login UI
- Organization, region, sector, and unit selection
- Dashboard with operation metrics, map mock, and recent activity
- Dark field-operations design system with reusable panels, chips, buttons, and app header

## Run

```sh
flutter pub get
flutter run
```

## Verify

```sh
flutter analyze
flutter test
```
